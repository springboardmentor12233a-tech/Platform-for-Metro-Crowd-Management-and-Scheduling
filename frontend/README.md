# 🚆 Metro Crowd Management & Scheduling System

## 📌 Project Overview

The Metro Crowd Management & Scheduling System is a full-stack web application developed as part of the Infosys Springboard Internship.

It provides a dashboard to monitor metro statistics such as:
- Total Stations
- Total Trains
- Total Schedules
- Crowd Status

The project uses cleaned datasets and demonstrates data preprocessing, backend API development, and frontend integration.
